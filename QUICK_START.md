### What You Need
- Chrome or Edge browser
- Working microphone

### How to Use

1. **Open the HTML file** in Chrome/Edge
2. **Allow microphone access** when prompted
3. **Click "Start Call"** (green button)
4. **Select speaker role** - Customer (blue) or Agent (purple)
5. **Click "Start Recording"** (cyan button) and speak
6. **Switch speakers** anytime by clicking Customer/Agent buttons
7. **Click "Export"** when done to download transcript

### That's It!

- Blue messages = Customer
- Purple messages = Agent
- Switch roles anytime during recording
- Export saves everything as a text file
